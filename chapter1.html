☰ HTML

HTML의 기초 개념 정리

1. 텍스트 태그

<strong> → 굵은 글자

<em> → 기울임

<del> → 취소선

<small> → 작은 글자

<sub> / <sup> → 기본 아래표기 및 위표기

2. 링크 & 이미지

<a href="#"> → 하이퍼링크(naver) 예시

<img src="이미지URL"> →

3. 리스트(List)

<ul> <li> → 순서 없는 리스트 예시

  • unordered item 1
  • unordered item 2

<ol> <li> → 순서 있는 리스트 예시

  1. ordered item 1
  2. ordered item 2

4. CSS 관련 기초

<p style="border: 2px solid black;"> → border 예시

<p style="font-size: 24px; font-weight: bold;"> →
큰 글씨 예시

<p style="text-align: center;"> → 가운데 정렬 예시

<p style="text-decoration: underline;"> → 밑줄 예시

<p style="text-indent: 20px;"> → 들여쓰기 예시

<p style="font-size: clamp(14px, 4vw, 24px);"> → clamp 예시
(화면 크기에 따라 달라지는 설정)
